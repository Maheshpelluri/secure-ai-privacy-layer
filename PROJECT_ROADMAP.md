# Project Roadmap

This roadmap mirrors the repository's phased direction and does not introduce
new scope.

## Phase 0

Architecture publication, research framing, and boundary definition.

## Phase 1

Core Engine: a minimal transformation and restoration engine for a single data
format.

## Phase 2

Provider Interface: integration with at least one public AI provider.

## Phase 3

Evaluation: reproducible testing of fidelity, latency, and failure modes.

## Phase 4

Prototype: a usable interface or reference implementation.

## Phase 5

Production: security review, compatibility management, and operational
hardening.
